# Hero font: Qurova

The hero section uses **Qurova Regular**. Qurova is not on Google Fonts.

To enable it, add the font files here:

- `Qurova-Regular.woff2` (preferred)
- `Qurova-Regular.woff` (optional fallback)

Then the hero headline will use Qurova. If the files are missing, the hero falls back to Montserrat.

You can download Qurova from the font’s official or licensed sources and place the files in this folder.
